# liri-node-app
A Node-based app similar to SIRI that searches OMDB, BandsInTown and Spotify APIs.

## Instructions
To use, enter the following `commands` in Terminal/Bash:

* `concert-this`
* `spotify-this-song`
* `movie-this`
* `do-what-it-says`

See the examples below for further instructions.
